EfficientNetを再現したい
Chainer Deep Learning
Lastmod: 2019-06-02

元ネタ: Tan and Le, “EfficientNet: Rethinking Model Scaling for Convolutional Neural Networks”, Arxiv:1905.11946, 2019.

リポジトリ

EfficientNetは強化学習によって設計された画像識別モデル.軽くて性能がいい(らしい).

公式のリポジトリを見ながら実装.パラメータ数は論文と同じ(目分量)になったし,順伝播と逆伝播が動くので次は学習コードを書く.

気がかりな点は,論文だと 224 -> 112 -> 112 -> 56 -> 28 -> 28 -> 14 -> 7 ->7と解像度が落ちていくのにこの実装(と公式実装)だと 224 -> 112 -> 112 -> 56 -> 28 -> 14 -> 14 -> 7 -> 7となっている点. しばらく待っていればどちらかに修正が入るはず.
